# Project notes for Claude

## MagicVille ↔ Scryfall ref mapping

**Do NOT assume that a MagicVille ref matches a Scryfall collector number.**
For example, MagicVille ref `sld2367` does not reliably correspond to Scryfall set `sld` collector `2367`.
Never try to parse or reconstruct a Scryfall printing from a MagicVille ref string.
